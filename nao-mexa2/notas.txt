Instalação do TOMCAT

File -> New -> Other... -> Server -> Server -> Next -> Tomcat V10.01 -> Next -> (Selecionar pasta) -> Next -> Finish

Guia inferior -> Aba servers -> Tomcat -> (Botão direito) -> Properties -> Switch Locations -> Apply and Close

Guia inferior -> Aba servers -> Clicar no tomcat -> Use tomcat Instalation -> Ctrl + S

Obs: Os passos 2 e 3 precisam ser feitos antes de startar o servidor pela segunda vez, se não, você terá de baixar o tomcat de novo.
